อาชีพรอง : (ดึง HM45 จากตอนที่ 1 มาแสดง)
สถานภาพการทำงาน : (ดึง HM46 จากตอนที่ 1 มาแสดง)
อุตสาหกรรม : (ดึง HM47 จากตอนที่ 1 มาแสดง)
รายได้จากอาชีพรอง : (ดึง HM_IN2 จากตอนที่ 1 มาแสดง)
รายได้จากการทำงาน
IN02 : รายได้/ ค่าตอบแทนจากการทำงานโดยได้รับเป็นค่าจ้าง/เงินเดือน
(รวม ค่าล่วงเวลา โบนัส ค่าทิป ค่าเปอร์เซ็นต์การขายสินค้าหรือบริการ ค่าครองชีพ ค่าช่วยเหลือบุตรและอื่น ๆ)
IN03 : รายรับสุทธิ จากการประกอบธุรกิจ อุตสาหกรรม การทำงานส่วนตัว หรือวิชาชีพที่ไม่ใช่การเกษตร
(หักค่าใช้จ่ายในการดำเนินการ)
IN04 : รายรับสุทธิ จากการประกอบการเกษตร หรือจากการให้บริการทางการเกษตร
(หักค่าใช้จ่ายในการดำเนินการ)
รายได้จากแหล่งอื่น ๆ ที่ไม่ใช่จากการทำงาน
IN05 : บำเหน็จ บำนาญ เบี้ยหวัด เงินสงเคราะห์ต่าง ๆ
(รวม เงินบำนาญ/บำเหน็จ จากกองทุนประกันสังคม)
IN06 : เงินชดเชยและ/หรือเงินทดแทนจากการออกจากงาน
IN07 : เงินช่วยเหลือ/สิ่งของที่ได้รับจากบุคคลอื่นนอกครัวเรือน
(รวม เงินจากญาติ/ไม่ใช่ญาติ ที่อยู่นอกครัวเรือน)
IN08 : เงินสงเคราะห์ผู้สูงอายุ และผู้พิการ รวมทั้งความช่วยเหลืออื่น ๆ จากรัฐและองค์การต่าง ๆ
IN08_1 : เงินสงเคราะห์เพื่อการยังชีพสำหรับผู้สูงอายุ
default = HM_IO08_1 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN08_2 : เงินสงเคราะห์สำหรับผู้พิการ
default = HM_IO08_2 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN08_3 : อาหารกลางวัน/อาหารเสริมฟรี (นม) สำหรับนักเรียน
default = HM_IO08_3 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN08_4 : การใช้จ่ายผ่านบัตรสวัสดิการแห่งรัฐ (ผู้มีรายได้น้อย)
default = HM_IO08_4 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN08_5 : เงินอุดหนุนเพื่อการเลี้ยงดูเด็กแรกเกิดจนถึงอายุ 6 ปี (พม.)
default = HM_IO08_5 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN08_6 : การช่วยเหลืออื่น ๆ จากรัฐและองค์กรต่าง ๆ
default = HM_IO08_6 (ตอนที่ 1) ถ้าไม่มีเป็น blank
รวมทั้งสิ้น
IN09 : เงินทุนการศึกษา
(รวม ทุนการศึกษาจากเงินของโรงเรียน)
default = HM_IO09 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN10 : รายรับจากการให้เช่าห้อง/เช่าที่ดิน และสินทรัพย์อื่น ๆ (โดยไม่ได้ประกอบเป็นธุรกิจ)
IN11 : ค่าลิขสิทธิ์และสิทธิบัตรต่าง ๆ
IN12 : ดอกเบี้ย/ผลตอบแทนจากเงินฝาก พันธบัตร เงินปันผลจากหุ้นและการลงทุนอื่น ๆ
IN13 : ดอกเบี้ย/ผลตอบแทนจากแชร์ และการให้กู้ยืมเงินแก่บุคคลอื่น/เอกชน
IN14 : มรดก พินัยกรรม ของขวัญ เงินซอง
(ไม่รวม ทรัพย์สิน)
IN15 : เงินได้รับจากการประกันสุขภาพ อุบัติเหตุ ไฟไหม้ ประกันชีวิต เงินฌาปนกิจศพ เงินได้จากประกันสังคม (ที่ไม่ใช่ประเภทสะสมทรัพย์) ทุกประเภท
(รวม เงินสงเคราะห์บุตรแรกเกิดจนถึงอายุ 6 ปี (ประกันสังคม)
default = HM_IO15 (ตอนที่ 1) ถ้าไม่มีเป็น blank
IN16 : รายได้อื่น ๆ
เช่น เงินถูกสลากกินแบ่งของรัฐ เงินรางวัล ค่านายหน้า (โดยไม่ได้ประกอบเป็นธุรกิจ) เงินได้จากการพนัน
รวมรายได้ทั้งสิ้น
